OK

127.0.0.1:6379> get myKey

Basic Redis commands:


SET: Set a value for a key.


SET mykey "Hello, Redis!"
GET: Get the value of a key.


GET mykey
DEL: Delete a key.


DEL mykey
EXISTS: Check if a key exists.


EXISTS mykey
KEYS: List all keys matching a pattern.


KEYS *
INCR: Increment the value of a key.


INCR mycounter
LPUSH: Push a value onto a list.


LPUSH mylist "value"
LRANGE: Get a range of values from a list.


LRANGE mylist 0 -1
HSET: Set a field in a hash.


HSET myhash field "value"
HGET: Get a field from a hash.


HGET myhash field
HGETALL: Get all fields and values from a hash.


HGETALL myhash
